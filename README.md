## ⚙️ How to Use Assets in Unity

If you plan to use the assets in your Unity projects, here’s the basic process:

1. Open Unity Editor on your PC.
2. Create a new project or open an existing one.
3. In Unity, go to **Assets > Import Package > Custom Package**.
4. Navigate to the folder where downloaded asset files are saved.
5. Select the asset files (usually `.unitypackage` files) and click **Open**.
6. Follow the prompts to import the files into your project.
7. The asset will now appear in your Unity Editor’s Project window under the Assets folder.

You can drag the asset from the Project window into your scene to use it.
